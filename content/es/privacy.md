Este sitio web no recoge datos personales, no utiliza cookies y no emplea servicios de terceros que rastreen la actividad del usuario.

El sitio se sirve mediante GitHub Pages. GitHub puede registrar datos técnicos básicos (como la dirección IP) únicamente con fines de seguridad y funcionamiento del servicio. Estos datos no son accesibles para el propietario del sitio web y no son utilizados para ningún tratamiento adicional.

Los enlaces para compartir contenido en redes sociales son simples enlaces; no cargan scripts de terceros ni transmiten información hasta que el usuario decide acceder voluntariamente a la plataforma correspondiente.

Si en el futuro este sitio incorpora formularios, cookies o servicios de terceros que impliquen tratamiento de datos personales, esta política será actualizada en consecuencia.
