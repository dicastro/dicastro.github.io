This website does not collect personal data, does not use cookies, and does not employ third-party services that track user activity.

The site is hosted through GitHub Pages. GitHub may log basic technical data (such as IP addresses) solely for security and service-operation purposes. This information is not accessible to the site owner and is not used for any additional processing.

The social-media sharing links on this website are simple hyperlinks; they do not load third-party scripts nor transmit any information until the user voluntarily clicks and accesses the respective platform.

If this website incorporates forms, cookies, or third-party services in the future, this policy will be updated accordingly.
